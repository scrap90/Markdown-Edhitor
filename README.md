# GlowEdit - プレミアム・マークダウンエディタ

GlowEdit は、HTML、CSS、Vanilla JavaScript のみで構築された、軽量・高速かつ美しいデザインのマークダウンエディタです。

サーバーの立ち上げやパッケージのインストールを必要とせず、かつ**インターネット接続が一切ない完全なオフライン環境**でも、ファイルをダブルクリックするだけで 100% 正しく動作するよう設計されています。

## 特徴

- 💻 **リアルタイム・ライブプレビュー**: `marked.js` を用いたマークダウンのパース、および `dompurify` による安全なサニタイズ処理。
- 🎨 **日本語化 & ツールチップ**: メニュー、ツールチップ、エラーメッセージ、統計情報まですべて日本語に対応。
- 📦 **完全ローカル化 (100% オフライン動作)**:
  - 外部ライブラリ（marked, DOMPurify, Prism）を `libs/` ディレクトリに内包。
  - Webフォント（Google Fonts/Material Icons）への依存を排除し、OS標準のシステムフォントおよび**インライン SVG アイコン**を採用。ネット環境のない屋外や機内でも完全に動作します。
- 🔄 **スクロール同期**: エディタとプレビューのスクロール位置をリアルタイムで双方向同期（オン/オフ切り替え可能）。
- 🖨️ **印刷・PDF出力 (`@media print`)**: ツールバーの「印刷」または `Cmd + P` (Mac) / `Ctrl + P` (Win) で動作。ヘッダーやエディタ領域を自動的に非表示にし、美しく装飾されたマークダウン文章のみを用紙やPDFとして出力可能。
- 💾 **自動下書き保存 (Auto Save)**: 入力したテキストと設定テーマは自動的に `localStorage` に保存され、誤ってブラウザを閉じても復元されます。
- 🛠️ **ドラッグ＆ドロップによる幅調整**: エディタとプレビューの境界線をドラッグして、表示幅を自由に調整できます。
- 📱 **レスポンシブ対応**: スマートフォンやタブレット表示の際は自動的に「エディタ」「プレビュー」のタブ切り替え式レイアウトへ移行します。

## 起動方法

### 1. ダブルクリックで起動（最も簡単、完全ローカル）
インターネットがオフラインの状態でも、プロジェクトフォルダ内の **`index.html`** ファイルをダブルクリック（ブラウザで直接開く）するだけで即座にエディタが起動します。

> [!NOTE]
> **`file://` 起動時のファイル保存の挙動について**
> ブラウザのセキュリティ制限により、`file://` プロトコル（直接ダブルクリック起動）で開いた場合は、元のファイルに直接上書き保存するAPI（File System Access API）が使用できません。
> そのため、「保存」や「別名で保存」をクリックした際は、自動的に**ブラウザのファイルダウンロード機能**を介して `.md` ファイルがローカルに保存されます。

### 2. ローカルサーバーを介した起動（ファイル上書き保存機能を使用する場合）
セキュリティ制限を解除し、ローカルの `.md` ファイルを「直接上書き保存」したい場合は、ローカルのセキュアコンテキスト（localhost）で実行してください。

```bash
# Python3を使用する場合の起動コマンド
python3 -m http.server 8000 & sleep 1 && open http://localhost:8000
```
起動後、ブラウザで `http://localhost:8000` を開くことで、ネイティブアプリのようにファイルの上書き保存が可能になります。

## ファイル構成

- `index.html` - 日本語化およびインラインSVGアイコンを内包したメイン骨組み。
- `style.css` - OS標準の日本語フォント最適化、スプリットレイアウト、カスタムスクロールバー、印刷用スタイル。
- `app.js` - エディタ本体の制御ロジック、`file://` 自動検知とフォールバック、キーボードショートカット。
- `libs/` - オフライン動作用の外部ライブラリファイル群（marked, DOMPurify, Prism）。
